Pros / cons at a glance

A · Stacked bar

+ Simple, minimal

+ Clean over-budget state (red bar)

− Percentages not explicit

− Can feel generic

B · Donut

+ Reads as "finance" immediately

+ Centre shows hero number

− More chrome / less mobile-native

− Over-budget state trickier (slice overflow)

C · Tinted tiles

+ Each line is self-contained

+ Tinted tiles match app language (Zaeli Noticed, Dashboard)

+ Mini bar shows proportion at a glance

− Doesn't show "total = 100%" as naturally

D · Labelled bar

+ % labels inside bar = fastest read

+ Amount chips below anchor dollars

+ Natural "Where your money goes" title

− Slightly busier than A

My pick

D (labelled bar + breakdown chips). Strongest one-glance read: the % inside each segment + dollar in the chip beneath means the user doesn't decode a legend or a donut. Feels warm (mint/sky/peach) and native to the Zaeli colour language. C is a strong runner-up if you want the visual to match other tinted tiles across the app.

Palette change summary

  • #059669 emerald → #2D7A52 deep green (primary action, progress fills)
  • #ECFDF5 emerald tint → #E6F7EF mint tint (light cards)
  • #34D399 mint mid → #B8EDD0 mint (accents, surplus)
  • #D1FAE5 → #C8F0DA (light borders)
  • #3B82F6 blue → #A8D8F0 sky (savings accent — matches My Space)
  • #EF4444 red → #FAC8A8 peach (over state — warm warning, not alarm)
  • Wordmark a+i → mint #B8EDD0
  • Over state: peach fill + #8A3A00 text (readable, warm)
